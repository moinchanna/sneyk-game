How to Play

01

Move

Use the Arrow keys or WASD on desktop. On mobile or touch devices, swipe in the direction you want the snake to slither.

02

Eat

Navigate toward the red food square. Each food eaten increases your length, boosts your score, and slightly accelerates the pace.

03

Survive

Avoid colliding with the container walls or your own growing tail. Crash, and the run is over. Good luck!

About

Sneyk is a modern retro tribute to the classic arcade and mobile Snake games of the late 90s. Reimagined with premium dark-mode visuals, smooth input controls, responsive scaling, and locally-synthesized sound effects.

This implementation is 100% client-side, open-source, and free of trackers, ads, or cookies. It is built strictly using TypeScript, lightweight modern CSS, and HTML5 Canvas.

Source code is hosted on GitHub. You can view the project, report bugs, or contribute via the GitHub repository.
